================================================================================
UTILISATION
================================================================================

Terminal 1 - Démarrer Ollama:
  $ ollama serve

Terminal 2 - Lancer l'agent:
  $ cd mon-agent-langgraph
  $ source venv/Scripts/activate
  $ python main.py

Terminal 2 - Utiliser l'agent:
  Vous: Bonjour!
  Agent: Bonjour! Je suis ici pour répondre à vos questions...

  Vous: Quel est la capitale de France?
  Agent: La capitale de la France est Paris...

  Vous: quit

================================================================================
PERFORMANCE
================================================================================

Première réponse: ~30-60 secondes (chargement)
Réponses suivantes: ~10-30 secondes (CPU)
Avec GPU NVIDIA: ~5-15 secondes
Avec GPU Apple: ~3-8 secondes

La première réponse est plus lente car le modèle doit se charger.

================================================================================
OPTIONS DISPONIBLES
================================================================================

1. UTILISER GEMMA 3 (quand téléchargement terminé)
   Mettre à jour .env: OLLAMA_MODEL=gemma3:latest
   Puis relancer: python main.py

2. UTILISER UN MODELE PLUS RAPIDE
   $ ollama pull mistral:latest
   Mettre à jour .env: OLLAMA_MODEL=mistral:latest
   Performance: Plus rapide (5-20s au lieu de 10-30s)

3. UTILISER ANTHROPIC CLAUDE
   Mettre à jour .env:
     LLM_PROVIDER=anthropic
     ANTHROPIC_API_KEY=sk-ant-v0-xxxxx...
   Avantages: Plus puissant, outils supportés, plus rapide

================================================================================
LIMITATIONS CONNUES
================================================================================

Ollama (Mode actuel):
  - Pas de support automatique des outils (météo, calcul)
  - Plus lent sur CPU
  - Réponses génériques sans outils spécialisés

Solutions:
  - Utiliser Mistral (plus rapide)
  - Utiliser Gemma 3 (meilleur)
  - Passer à Claude (outils complets)
  - Ajouter GPU pour accélérer
