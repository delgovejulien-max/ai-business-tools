Exemples de questions:

  "Quelle est la météo à Lyon et Marseille?"

  2. calculate(expression)  → Calcule des maths

Gemma 3 décide automatiquement quel outil utiliser!

================================================================================
TROUBLESHOOTING RAPIDE
================================================================================

"Connection refused"
  → ollama serve n'est pas actif dans Terminal 1

"Model not found"
  → Attendre que Gemma 3 finisse de télécharger
  → Vérifier: ollama list

"Module not found"
  → Pip install -r requirements.txt

"Response very slow"
  → Normal sur CPU
  → Utiliser mistral:latest (plus rapide)
  → Ou installer GPU support
