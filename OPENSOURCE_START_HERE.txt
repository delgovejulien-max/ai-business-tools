================================================================================
TROUBLESHOOTING
================================================================================

"Connection refused"
→ Ollama n'est pas actif
→ Solution: ollama serve dans un terminal différent

"Model not found"
→ Gemma 3 n'est pas téléchargé
→ Solution: ollama pull gemma3:latest

"Module not found langchain"
→ Dépendances non installées
→ Solution: pip install -r requirements.txt

"Response very slow"
→ Utiliser CPU sans GPU
→ Solutions:
   1. Utiliser modèle plus petit (mistral)
   2. Installer GPU support (NVIDIA CUDA)
   3. Ajouter plus de RAM
