  OLLAMA_MODEL=gemma3:latest

================================================================================
TROUBLESHOOTING
================================================================================

"Connection refused"
  → Ollama n'est pas actif
  → Solution: Assurez-vous que "ollama serve" est lancé

"Model not found"
  → Gemma 3 n'est pas trouvé
  → Solution: ollama list (vérifier si apparaît)
  → Sinon: ollama pull gemma3:latest

"Module not found langchain_ollama"
  → Dépendances non installées
  → Solution: pip install -r requirements.txt

"Response very slow"
  → CPU sans GPU
  → Solutions:
     1. Utiliser modèle plus petit: ollama pull mistral
     2. Installer GPU support (NVIDIA CUDA)
     3. Ajouter plus de RAM (8+ GB recommandé)
