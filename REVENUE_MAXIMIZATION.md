## THE DECISION: 3-Tier Pricing (Not 4)

**Eliminating Free Tier Bloat**

```
CURRENT (Mistakes):
FREE ($0)    -> 5 tools/month (too generous)
PRO ($29)    -> Unlimited (cannibalization risk)
TEAM ($99)   -> 10 people (underutilized)
ENTERPRISE   -> Custom pricing (slow deal cycles)

OPTIMIZED (Revenue-Focused):
FREE ($0)    -> 1 tool/month (hook only)
PRO ($29)    -> Unlimited tools + exports (main tier)
TEAM ($99)   -> 10 people + collaboration (expansion)
ENTERPRISE   -> $999+ (on request)
```

**Rationale:**
- Free tier = customer acquisition (not revenue)
- Free tier users consuming 5 tools are potential PRO customers
- Limited free = upgrade trigger
- Concentrated value = 3 tiers, not 4

**Expected Revenue Impact:** +40% conversion (2% → 2.8%)

## DECISION 1: Focus on PRO Tier Conversion

**Target: 10% of signups -> PRO ($29/month)**

### Week 1-2: Create the Hook
```

- Users see TAM and opportunity
- Can't try competitors/pricing/GTM
- Natural upgrade trigger
```

### Week 2-4: Make PRO Irresistible
```
PRO INCLUDES:
✓ Unlimited tools access
✓ PDF/Excel exports
✓ Shareable links
✓ AI recommendations
✓ 3 co-founders
✓ Email support

MESSAGING: "Everything you need to pitch investors"
PRICE: $29/month (easy mental commitment)
TARGET: Serious founders (will convert)
```

### Week 3-4: Deploy Upgrade Triggers

**Trigger 1: Limited Usage**
```
After using 1st tool:
"Want to try competitors/pricing/launch analysis?"
[Upgrade to PRO] [Not now]
```

**Trigger 2: Export Attempt**
```
"Want to download your analysis?"
[Free: View only] [PRO: Download PDF/Excel]
```

**Trigger 3: Team Invite**
```
"Invite your co-founder to review?"
[Free: Can't] [PRO: Invite 3 people]
```

**Trigger 4: After 7 Days**
```
Email: "You've completed 1 analysis.
Ready to explore full toolkit?
Try PRO free for 7 days."
```

**Expected Result:** 5-10% of free users convert to PRO
