================================================================================
TROUBLESHOOTING
================================================================================

"Ollama not responding"
  → Check Terminal 1 is running: ollama serve
  → Port 11434 must be available

"Connection refused"
  → Make sure ollama serve is active before running ceo_service.py

"Python not found"
  → Make sure you activated venv first:
    venv\Scripts\activate

"Unicode errors"
  → Normal on Windows - the agent still works fine

"Agent seems slow"
  → First response takes 30-60s (model loading)
  → Subsequent responses: 10-30s
  → This is normal on CPU
