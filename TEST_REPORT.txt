HOW TO USE
================================================================================

Terminal 1:
  ollama serve

Terminal 2:
  cd mon-agent-langgraph
  source venv/Scripts/activate
  python main.py

Then ask questions:
  Vous: Quel est la capitale de France?
  Agent: [Response from Gemma 2b]

  Vous: quit
